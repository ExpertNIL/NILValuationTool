# ExpertNIL Landing Site (v1)

## What’s included
- `index.html` — animated techy landing page
- `request-access.html` — request access form UI (front-end only)
- `assets/` — logo + checkmark + font

## How to run locally
Just open `index.html` in a browser (Chrome/Safari/Edge).

## Add your demo video
Open `index.html` and find:
`const DEMO_YOUTUBE_EMBED_URL = "";`

Paste a YouTube *embed* URL (unlisted is fine). Example:
https://www.youtube.com/embed/VIDEO_ID?mute=0

The site will add autoplay when the modal opens.

## Deploy (GitHub Pages quick)
1) Create a repo, upload these files
2) Settings → Pages → Deploy from branch (main / root)
3) Your site will appear at `https://<username>.github.io/<repo>/`
